Teaser Component

A teaser component consists of an image, title and description.

Read More

The Teaser Component combines an Image, Title, Description, and Link. On this page there are 2 styles being applied. The second is a card type style. The Hero style needs a little bit of Javascript to copy the image 'src' attribute as an in-line style background-image.

*Note the Teaser Component is currently in development, expected to be released early Spring.
