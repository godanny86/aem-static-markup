Image Component

The image component displays an image. Optionally a link can be added to surround the image. When deployed within AEM smart loading of optimal rendition is enabled. Notice that in the above example in Desktop there are three images side by side. In the tablet Breakpoint the third image stretches full width for the container and in Mobile breakpoint all three images strech the full width.
